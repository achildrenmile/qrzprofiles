# OE8YML QRZ Page

Custom HTML bio page for [OE8YML on QRZ.com](https://www.qrz.com/db/OE8YML).

## Features

- Dark theme with amber accents
- Pulsing ON AIR indicator (CSS animation)
- Morse code visual separators
- Responsive card-based layout
- Band/mode badges with color coding
- Station equipment showcase
- Digital IDs with Brandmeister link
- SOTA, POTA, ClubLog iframes
- HamAlert recent spots
- Solar propagation widget
- OERadio.at integration
- Club & membership logo grid
- QSL & logging info

## Usage

1. Open `oe8yml_qrz_page.html`
2. Copy the entire contents
3. Go to QRZ.com → Edit bio → Source mode
4. Paste and save

## Constraints

Built within QRZ bio editor limitations:
- No JavaScript
- No external fonts
- No external CSS
- CSS `<style>` block + inline styles only
- Iframes from approved domains only
