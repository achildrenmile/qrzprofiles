# QRZ Profiles

Custom HTML bio pages for QRZ.com.

## Pages

### [OE8YML](https://www.qrz.com/db/OE8YML) — `oe8yml_qrz_page.html`

Dark theme with amber accents. Radio shack dashboard style.

- Pulsing ON AIR indicator, morse code separators
- Band/mode badges, station equipment showcase
- Digital IDs with Brandmeister link
- SOTA, POTA, ClubLog, HamAlert iframes
- Solar propagation widget, WebRX.AT
- OERadio.at featured section
- OE8JOTA / Next Generation
- Club & membership logo grid, QSL info

### [OE8YYY](https://www.qrz.com/db/OE8YYY) — `oe8yyy_qrz_page.html`

Light alpine theme with rose/mauve accents. Outdoor magazine style.

- Mountain gradient hero with SOTA Active badge
- WWAYL (Young Ladies World Wide Award) iframe
- SOTA activations iframe
- Summit photo gallery
- OE8JOTA / Next Generation

## Usage

1. Open the `.html` file for the callsign
2. Copy the entire contents
3. Go to QRZ.com → Edit bio → Source mode
4. Paste and save

## Constraints

Built within QRZ bio editor limitations:
- No JavaScript
- No external fonts or CSS
- CSS `<style>` block + inline styles only
- Iframes from approved domains only
